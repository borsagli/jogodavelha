# Jogo da Velha em C#

Projeto desenvolvido em **C#** com o objetivo de praticar matrizes, arrays e lógica de jogo.

## Projeto
Implementação do clássico jogo da velha utilizando estruturas de dados para controle do tabuleiro e validação de jogadas.

## Tecnologias
- C#
- .NET
- Visual Studio

## Aprendizados
- Uso de matrizes 
- Estruturas condicionais
- Organização da lógica do jogo
